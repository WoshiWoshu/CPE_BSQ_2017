# CPE_BSQ_2017

First year mini project consisting of finding the largest possible square on a board while avoiding obstacles. The board is represented by a file passed as the program’s argument. “.” (representing an empty place) and “o” (representing an obstacle). “.” is replaced by “x” to represent the largest square founded. It is a console program in C language.

To launch the C program : 1) -> Make 2) -> ./bsq example_file
